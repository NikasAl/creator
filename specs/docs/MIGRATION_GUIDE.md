# Миграция со старых скриптов

## Старые скрипты (можно удалить):
- `run_full_pipeline.sh`
- `run_full_pipeline LemEng.sh` 
- `run_full_pipeline_Lomonosov.sh`

## Новые скрипты:
- `run_pipeline.sh` - основной скрипт
- `run_stalin.sh` - замена для `run_full_pipeline.sh`
- `run_lem.sh` - замена для `run_full_pipeline LemEng.sh`
- `run_lomonosov.sh` - замена для `run_full_pipeline_Lomonosov.sh`

## Что изменилось:

### Исправления:
1. **Исправлена ошибка "No module named 'text_processors'"** - все Python скрипты теперь используют абсолютные пути
2. **Добавлено создание HTML файла** - после создания промо-описания с паузой для редактирования
3. **Улучшена обработка ошибок** - более информативные сообщения

### Улучшения:
1. **Единая кодовая база** - вся механика в одном файле
2. **Конфигурационные файлы** - параметры вынесены в отдельные файлы
3. **Легкость добавления новых проектов** - достаточно создать конфигурационный файл
4. **Лучшая структура проекта** - конфигурации в отдельной папке

## Использование:

### Вместо:
```bash
./run_full_pipeline.sh
```

### Теперь:
```bash
./run_stalin.sh
```

### Или напрямую:
```bash
./run_pipeline.sh configs/stalin.conf
```
